# 3.28 修改说明

## 本次完成内容

### 1. 多张答卷图片分组上传
- 前端支持一次选择多张图片上传。
- 可按“每位学生页数”自动分组，将连续图片视为同一位学生的答题纸。
- 每组答卷可点击卡片弹窗预览，方便人工确认分组是否正确。

### 2. 成绩列表页面
- 右侧结果区改为成绩列表展示。
- 列表展示字段目前为：`姓名`、`科目`、`得分`、`时间`。
- 每条记录支持：`查看答卷`、`查看详情`、`删除`。

### 3. 查看详情弹窗
- 识别摘要、阅卷总评、分页摘要不再直接堆在主页面。
- 这些内容被移动到“查看详情”弹窗中展示。
- 弹窗中保留总分信息、识别内容和分页摘要。

### 4. 学生成绩删除
- 每个学生成绩记录都可单独删除。
- 删除时会同时删除该记录关联的本地答卷图片文件。

### 5. 姓名与科目识别优化
- 上传后的答卷图片会先做方向归正，横向图片会自动转为竖向。
- 对第一页顶部区域进行单独裁切，用于优先识别 `姓名` 和 `科目`。
- 新提示词明确要求：
  - 姓名必须是学生真实姓名
  - 不得把学院、专业、班级误识别成姓名
  - 科目优先从试卷标题中提取

### 6. 手动修正姓名和科目
- 在“查看详情”弹窗中新增了姓名、科目编辑框。
- 如果模型识别错误，可直接手动修改并保存。
- 保存后成绩列表会同步刷新。

### 7. 布局与样式调整
- 左右两张主卡片调整为更均衡的布局。
- 成绩列表区域加宽，尽量减少横向滚动。
- 删除了“页数”“判定”两列，让列表更紧凑。
- 修复了三个操作按钮重叠的问题。
- 调整了上传页数说明区域的排版，避免文案拥挤换行难看。

## 当前使用方式

1. 上传多张答卷图片。
2. 设置“每位学生页数”。
3. 系统按顺序自动分组生成多位学生的答卷。
4. 点击“生成成绩列表”。
5. 在成绩列表中查看每位学生的姓名、科目和得分。
6. 如识别有误，可进入“查看详情”手动修正姓名和科目。

## 说明

- 新的姓名识别优化主要对之后新生成的记录生效。
- 已经存储的旧记录不会自动重新识别。
- 若旧记录姓名有误，可直接在详情弹窗中手动修改。
